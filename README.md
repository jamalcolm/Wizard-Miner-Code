# Wizard-Miner-Code
Boulder Dash inspired game developed using Unity C#

**Controls (Keyboard):** 
Arrow Keys/WASD - Move
Space - Action (dig/collect diamonds without moving)

**Controls (Xbox 360 Controller):** 
Left stick - Move
B - Action
